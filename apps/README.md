先に music-lab.html でキーを設定しておくか、ブラウザコンソールで直接設定できます。

// コンソールで実行

## ✦ COMPOSE タブ

作曲の起点。**Step Input（左）** と **Quick Composer（右）** が同時表示されます。

-----

### Step Input（左ペイン）

音符をキーボードで高速入力するモジュールです。

#### キーマッピング

|キー|音名||キー|音名  |
|--|--||--|----|
|A |C ||H |A   |
|W |C#||U |A#  |
|S |D ||J |B   |
|E |D#||K |C+  |
|D |E ||Z |1音削除|
|F |F ||  |    |
|T |F#||  |    |
|G |G ||  |    |
|Y |G#||  |    |

#### 操作フロー

```
1. 「対象トラック」を選択（新規 or 既存）
2. 「オクターブ」「音価」を設定
3. A〜K キーを押す → 画面下部にノートが積まれる
4. Z キーで1ノート削除（Undo）
5. [▶ COMMIT] ボタン → 選択トラックの末尾に追加
```

#### 音価の目安

|設定  |長さ       |
|----|---------|
|1/16|0.25拍    |
|1/8 |0.5拍     |
|1/4 |1拍（デフォルト）|
|1/2 |2拍       |
|1/1 |4拍       |

### Quick Composer（右ペイン）

コード進行を視覚的に組み立てるモジュールです。

#### セクション管理

- 上部の `Verse 1` `Chorus` `Bridge` ボタンでセクションを切替
- `＋` で新しいセクションを追加（名前を入力）
- 各セクションに独立したコード進行を持てる

#### コードパレットの使い方

```
1. 「キー」セレクターでキーを選択（C/D/E/F/G/A/B/Am/Dm/Em）
2. パレットに選択キーのダイアトニックコードが自動表示
3. コードをクリック → 進行タイムラインに追加 + 試聴
```

#### カスタムコード追加

```
1. 「ルート」グリッドで根音をクリック
2. 「タイプ」から和音タイプを選択
   (maj / min / 7 / maj7 / min7 / dim / aug / sus2 / sus4 / 9 / add9 / 6 / dim7)
3. 「＋ パレットに追加」ボタン
```

#### 進行タイムライン操作

|操作          |結果                 |
|------------|-------------------|
|コードチップをクリック |試聴                 |
|ドラッグ & ドロップ |順番を入れ替え            |
|`×` ボタン     |削除                 |
|`✦ 提案` ボタン  |キーに合う4コードを自動提案     |
|`▶ 試聴` ボタン  |進行全体を順番に再生         |
|`→ ROLL` ボタン|コード進行を ROLL タブに書き出し|

## 🎹 ROLL タブ（Piano Roll）

音符を直接配置・編集するメインエディタ。

### ツールバー

|ボタン    |ショートカット  |動作              |
|-------|---------|----------------|
|✏ PEN  |`P`      |クリックで音符配置       |
|⊹ SEL  |`S`      |音符を選択・移動        |
|✕ ERASE|`E`      |クリックで削除         |
|グリッド   |—        |スナップ解像度（1拍〜1/16）|
|ZOOM   |Ctrl+ホイール|横方向の拡大縮小        |

### 音符の操作

|操作                    |結果       |
|----------------------|---------|
|グリッドをクリック（PENモード）     |音符を配置    |
|クリックしたままドラッグ          |音符の長さを伸ばす|
|音符をドラッグ（SELモード）       |上下左右に移動  |
|音符の右端をドラッグ            |長さを変更    |
|音符をクリック（ERASEモード）     |削除       |
|`Delete` / `Backspace`|選択ノートを削除 |

### トラック管理

- `+ MEL` / `+ CHD` / `+ BAS` で新規トラック追加
- 「トラック」セレクターで編集対象を切替
- 左端のピアノキーをクリック → 試弾

### ピアノロールの範囲

- C1（MIDI 24）〜 C8（MIDI 108）= 85音
- 縦: ピッチ、横: 時間（拍）
- Ctrl + ホイールでズーム

## ⊞ ARR タブ（Arrangement View）

曲全体の構成（イントロ〜アウトロ）を管理します。

### デフォルト構成

```
Verse 1 → Chorus → Verse 1 → Chorus → Bridge → Chorus
```

### 操作

|操作         |結果                  |
|-----------|--------------------|
|ブロックをドラッグ  |順番を入れ替え             |
|`×` ボタン    |そのブロックを削除           |
|`＋ セクション追加`|セクション番号を指定して追加      |
|`→ ROLLに展開`|全セクションをROLLに順番通り書き出し|


> **注意:** `→ ROLLに展開` を実行すると既存のROLLデータはクリアされます。確認ダイアログが出ます。

### セクションとの関係

- COMPOSEタブの各セクション（Verse/Chorus/Bridge）にコード進行を設定
- ARRタブでその順番を組み立てる
- `→ ROLLに展開` で時系列のMIDIデータに変換

## 🤖 AI タブ（AI Region Generator）

指定範囲にAIでノートを生成して挿入します。

### 使い方

```
1. モデルを選択（CLAUDE / GEMINI / OPENAI）
2. 「開始小節」と「小節数」で生成範囲を指定
3. 「対象トラック」を選択（新規 or 既存）
4. 「プロンプト」に生成指示を日本語で入力
5. 「パート種類」と「ジャンル」を選択
6. [✦ AI生成] ボタン
7. 右ペインにプレビューが表示される
8. [✓ ROLLに追加] で確定
```

### プロンプト例

```
Amキー、切ないサビメロディ、16分音符多め
シティポップらしいコード進行、4小節
ルート中心のシンプルなベースライン
夏の終わりをイメージしたカウンターメロディ
```

### 対応モデル

|GEMINI|`ml_gemini`           |無料枠あり・高速     |

## ⬇ MIDI エクスポート

トップバーの `⬇ MIDI` ボタンで全トラックをエクスポートします。

### 仕様

|項目    |内容                         |
|------|---------------------------|
|フォーマット|Standard MIDI File Format 1|
|PPQ   |480                        |
|トラック構成|テンポトラック + 各トラック（ch0〜ch15）  |
|ファイル名 |`プロジェクト名-タイムスタンプ.mid`      |

### GarageBand への取り込み手順（iPhone）

```
1. MIDI ボタン → .mid ファイルをダウンロード
2. iPhone の「ファイル」アプリを開く
3. .mid ファイルを長押し →「別のアプリで開く」
4. GarageBand を選択
5. Tracks ビューに各トラックが読み込まれる
```

## 推奨ワークフロー

### パターン A: コード先行

```
1. COMPOSE → キーを設定 → コードパレットで進行を組む
2. COMPOSE → 「→ ROLL」でコードをROLLに送る
3. ROLL → メロディ/ベーストラックを追加して打ち込み
4. DRUM → パターンを組んで「→ ROLL」
5. MIDI エクスポート → GarageBand
```

### パターン B: ステップ入力先行

```
1. COMPOSE → Step Input でメロディを入力 → COMMIT
2. AI タブ → コード進行をAI生成 → ROLLに追加
3. AI タブ → ベースラインをAI生成 → ROLLに追加
4. DRUM → パターン作成 → ROLL
5. MIDI エクスポート
```

### パターン C: AI フル生成

```
1. AI タブ → メロディ生成（8小節）
2. AI タブ → コード生成（同じ範囲）
3. AI タブ → ベース生成
4. ARR → セクション構成を組む → ROLLに展開
5. DRUM → ドラムパターン追加
6. MIDI エクスポート → GarageBand
```
